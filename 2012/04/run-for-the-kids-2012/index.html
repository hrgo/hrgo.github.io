Run for the Kids 2012
1 Apr 2012
2 minutes read

올해의 첫 Fun Run. Run for the Kids. 한번도 뛰어보지 않은 15km. 많이 쫄았는데.. Yay! 완주!

Course Map.

경건한 마음으로 준비해 놓은 나의 러닝킷.

두시간전에 아침을 거하게 먹고, 가는 길에 커피한잔  마시고 느긋하게 행사장 도착.  33000명이 참가하는 멜번의 큰 마라톤 행사다.

최지영이 응원단을 자청하여 동행했다. 출발선.

출발선에서. 나는 2004년에 처음 맞춘 “Happy Running 흐름”이 새겨진 나의 첫 유니폼을 입고 달렸다. 날씨는 좋았는데 최지영의 더러운 핸드폰 카메라 렌즈덕에 안개낀 날씨가 되심.

행사장을 나와 City Link Domain Tunnel 진입.

아직 초반이라 가벼운 발걸음으로 터널을 달리는 중. 소리 지르고 모두 신났다.  재밌긴 한데 GPS도 안잡히고 환기도 안되고. 터널 빠져나온 지점이 3km 되시겠다.

6km 지점. Bolte Bridge 램프 도는 중. 요 꼬맹이들 달리는거 보고 신기. 마라톤 꿈나문가?

약 8km 지점. Bolte Bridge. 잘못 진입해서 어쩔 수 없이 딱한 번 달려 본 유료도로. 오늘은 전세 내서 발로 달린다. 오르막이라 살짝 힘들었던 구간.

이후엔 사진이 없다. 9km를 좀 지나 파워젤을 하나 까먹고 힘을 내서 달리다가 11km지점에서 City로 진입하는 Collins Street의 오르막을 보고 욕이 절로 나왔다. 이때부터 발바닥의 물집이 쓰라리기시작했지만 City로 들어와서 인파가 많아지고 구경꾼도 늘어나 덩달아 힘을 내서 달릴 수 있었다. 마지막엔 어라 벌써 다왔네? 라고.

완주 후 post-face area. 핸드폰 시계로 약 1:38 달리심. 전화가 안터져서 일행들 찾느라 난리.

Finish line.

총평.

여타의 Fun Run이 보통 5km, 10km, Half 의 코스를 운영하는 반면 이 행사는 14.38km라는 이런 애매한 거리의 코스를 단지 City Link를 달릴 수 있다는 이유를 내세워 고집하고 있다. 좀 애매한 거리긴 하지만 10km는 좀 만만하고 Half는 좀 부담스런 나같은 사람이 도전해 볼 수 있는 코스일 듯.

City Link를 달려 보는 것은 좋으나, 시작하자마자 달려야 하는 2km길이의 터널, 구간 구간의 병목현상은 그렇다 쳐도 3만명이 넘는 많은 인원에 비해 미흡한 행사운영이 에라다. Run Melbourne에 비해 형편없는 행사 지원. Finish line의 급수를 예로들면 보통 마라톤 행사에서 끝나고 나눠주는 각종 음료와 간식들은 고사하고 물 한잔도 줄을 서서 컵에 받아 마셔야 했던 어이없는 상황. 내년에 또 하라고 한다면? 글쎄. 내년엔 Half를 달려야 하지 않겠니?

어쨌든 도전 성공. 4월 멋지게 Kick off!   사기가 하늘을 찌를 듯. ㅋㅋㅋ

  • 4월 Run for the Kids (14.38km)
  • 7월 Run Melbourne (10km)
  • 11월 Melbourne Marathon (Half)

올해 계획한 달리기 계획중 첫번째를 무사히 성공. 7월 10km는 그야말로 Fun Run이 될테고 11월의 Half에 쫌 빛이 보인다. 히히히.
