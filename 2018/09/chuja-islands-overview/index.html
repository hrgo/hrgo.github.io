추자도 Overview
22 Sep 2018
One minute read

개요

  • 제목 : 추자도 백패킹
  • 언제 : 2018년9월23일 ~ 2018년9월25일 (2박3일)
  • 동행 : 태정, 태욱, 창용, 인호, 아름, 경남

일정

  • 1일차 : 제주항 집결, 추자도로 이동. 상추자항에서 점심식사 후 등대전망대 찍고 묵리 마을로 이동. 예초리 왕복 버스투어. 해변에서 야영
  • 2일차 : 묵리에서 버스타고 모진이 해변으로 이동, 카페에 배낭 맡겨두고 하추자 트레킹, 모진이 해변에서 저녁 먹고 돈대산에서 야영
  • 3일차 : 돈대산 하산, 상추자로 이동. 식당에 배낭 맡겨두고 나바론길 트레킹, 점심 먹고 배타고 제주로.
