Overview

  • 기간 : 2002년 9월 (3박5일)
  • 장소 : 태국 방콕, 파타야
  • 동반자 : 빵래

History

  • 2002.6 여름 휴가를 어디로 갈것인가. 꿈에 부풀다.
  • 2002.7 개떡같은 회사에서 여름 휴갈 3일밖에 안준다고. 하루를 캐리비안베이 가는데 쓰기로 한 나는 갈등에 휩싸인다.
  • 2002.7.30 여름휴가를 4가지 방안을 두고 머릴싸맴. 영등포에서 타이 여행책자를 사면서 태국쪽으로 기울기 시작한다.
  • 2002.8월 2,3 주 타이 관련 싸이트를 써핑 하고, 여행사마다 9월 초 방콕 항공권 문의. 졸라비쌈.(50만원대)
  • 2002.8.18 동반자 나타남
  • 2002.8.19 일정을 박음. 추석을 성수기로 치므로 놀토를 끼고 항공권이 가장 싼 9월초로. 9.5~9.9 (4박5일)
  • 2002.8.20 항공권 경매 싸이트에서 항공권 예매. www.8226.co.kr
  • 2002.8.23 헬로 태국 책자삼. 하고싶은것, 보고싶은것들의 윤곽이 드러나고 있음.
  • 2002.8.26 하고싶은 것들을 나열. 굵은 가지를 그려본다. (방콕1.5 깐짜나부리1 꼬사멧 1)
  • 2002.8.30 일정 변경. 꼬사멧 대신 파타야. 꼬사멧 너무 멀고 심심할껏이야.
  • 2002.9.2 항공권 발권신청.(354,600 + TAX = 379,600 원) OX301 1225 ICN 1625 BANGKOK OK OX300 0100 BANGKOK 0820 ICN OK
